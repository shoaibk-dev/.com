## List opening ports or listening UNIX sockets

In Linux, you can use `netstat` to list all opening ports and listening UNIX sockets

```bash
$ sudo netstat -npl
```

Tip to remember command : **net**work **stat**istics - **n**atual **l**anguge **p**rocessing
