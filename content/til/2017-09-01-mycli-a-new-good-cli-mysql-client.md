## Mycli : a new good cli MySql Client

This tool is written in Python with super cool features (auto-complete and colors).

**Worth a shot !**

Install

```bash
$ pip install mycli
```

Usage

```bash
$ mycli -h 127.0.0.1 -P 3306 -u root
```
