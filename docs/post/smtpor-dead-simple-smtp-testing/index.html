SMTPor : Dead simple SMTP testing tool

Never need a bigass mail client to test your SMTP config !

Demo

TL;DR

Using Docker to run

$ docker run -d -p 3000:3000 khanhicetea/smtpor

Then open http://localhost:3000 to try a shot !

Install & Start

$ npm install
$ npm start
